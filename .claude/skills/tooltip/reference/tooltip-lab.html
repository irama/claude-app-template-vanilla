Anchored — the default

Icons, buttons, chips. The tip pins to the element and stays put while hovered or focused; it never chases the mouse. Every chip declares a preferred side; watch resolved match it when there's room.

Truncated text — tip only when clipped

onlyIfTruncated suppresses the tooltip when the text already fits — no redundant tips. Resize the window narrower to clip more rows and watch tips appear. Each row is keyboard-focusable.

03

Collision lab — shift & flip inside a boundary

These anchors pass boundary: arena — the tip must stay inside this box, not just the screen. Anchors are pinned flush to its edges to force overflow. The engine shifts along the cross-axis for small clips and flips to the opposite side when the preferred side won't fit the boundary. (Re-anchor to a perpendicular side is the last resort when both opposite sides fail.) The collision cell tags which fired.

04

Cursor-following — opt-in, for regions

One large hover surface, no single anchor. Pass mode:'follow' and the tip tracks the pointer, offset from it, updates throttled to requestAnimationFrame, and still stays on-screen. Keyboard can't reach a moving target, so the same data is mirrored in the readout below — the accessible alternative.

05

Keyboard & dismissal

Tab through — tips appear on focus, not just hover. First tip in a burst waits the full delay; the rest skip it (group warm-up), so scrubbing a toolbar feels instant.

06

The API

One factory. Declarative via data-* attributes for the common case, or imperative for full control.

target
Element the tip belongs to (or a [data-tooltip] scan).
content
String, {title, sub}, or a DOM node.
placement
top · bottom · left · right. Preferred only — fallback is automatic.
mode
anchored (default) · follow.
showDelay
ms hover-in. Default 420.
hideDelay
ms hover-out. Default 90.
offset
px gap from anchor/cursor. Default 12.
hoverable
Let the pointer enter the tip (WCAG). Default false.
onlyIfTruncated
Suppress unless the text is clipped.
// declarative — the 90% case
<button data-tooltip="Archive">🗄</button>
<div data-tooltip="…" data-tooltip-truncate>…</div>

// imperative — anchored
Tooltip.attach(btn, {
  content: { title: "Delete", sub: "30-day recovery" },
  placement: "bottom",
})

// imperative — cursor-following region
Tooltip.attach(canvas, {
  mode: "follow",
  content: (e) => readAt(e.clientX, e.clientY),
})
